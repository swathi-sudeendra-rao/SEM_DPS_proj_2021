# SEM_DPS_proj_2021

## client-server socket programming using pthreads

1. To run server.c:

    i) gcc server.c -pthread

    ii) ./a.out

2. To run multiple clients using client.c:

    i) gcc client.c -pthread
    
    ii) ./a.out client_name
